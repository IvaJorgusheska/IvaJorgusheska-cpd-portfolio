Robotics and Autonomous Systems

Multi-Agent Reinforcement Learning for Cooperative Autonomous Navigation and Pursuit in Drone Swarms

Developed a complete Multi-Agent Reinforcement Learning (MARL) framework from the ground up, including a custom simulation environment, modular PPO-based training pipeline, and visualization tools for real-time episode rendering. Extended the baseline PPO algorithm to handle partial observability and enable stable cooperative behavior among multiple autonomous drones. The framework was successfully deployed on physical Crazyflie 2.1 drones, demonstrating efficient real-world performance and robust swarm coordination. This project highlights the potential of lightweight, general-purpose algorithms for scalable and resource-efficient multi-agent robotics.

Deep Learning in Cognitive Robotics – UAV Applications and Vision Models

I conducted an extensive research study on the integration of Deep Neural Networks (DNNs) in autonomous Unmanned Aerial Vehicles (UAVs), producing a comprehensive, survey-style analysis of the latest advances in deep learning for robotic vision, navigation, and decision-making. The work critically reviewed state-of-the-art architectures such as CNNs, LSTMs, and reinforcement learning frameworks, assessing their roles in obstacle avoidance, surveillance, and multi-UAV coordination. I evaluated both technical and ethical limitations in current research and identified emerging trends in lightweight and adaptive DNN models for real-time aerial autonomy. Complementary experiments using MLP, CNN, and ResNet-18 on the CIFAR-100 dataset demonstrated practical understanding of network optimization and performance benchmarking.

Artificial Intelligence and Machine Learning

Machine Learning Laboratory – Linear Classification and Optimization Analysis

This lab focused on the mathematical foundations of machine learning, particularly linear classification and optimization through gradient descent. I derived the gradient of the hinge loss function with L2 regularization step-by-step, expressed it in vectorized form, and formulated the weight update equation analytically. The study examined how the learning rate affects convergence, cost reduction, and classification accuracy across training and testing sets. In the second part, I applied these principles to a neural network model for air quality prediction, analyzing optimizer performance (SGD vs. ADAM) and validating convergence behavior. The work demonstrated strong understanding of mathematical modeling, derivative computation, and optimization dynamics in supervised learning.

Features for Estimating Autonomous Vehicle Poses

Implemented and analyzed feature-matching algorithms for visual odometry (VO) in autonomous vehicle navigation using the KITTI dataset. Developed three matching strategies—distance thresholding, nearest neighbour, and nearest neighbour distance ratio (NNDR)—to estimate vehicle trajectories through sequential image analysis. Using SIFT-based keypoint detection and OpenCV’s BFMatcher, I compared feature similarity metrics to improve pose estimation accuracy. The work demonstrated how robust feature correspondence underpins reliable VO performance, particularly in complex, real-world motion scenarios. This project strengthened my understanding of computer vision, geometric reasoning, and the link between low-level image features and high-level motion inference in autonomous systems.

Reversi Game Agent – Minimax with Alpha–Beta Pruning

In this project, I implemented an intelligent Reversi (Othello) game agent in Java, designing a minimax search algorithm enhanced with alpha–beta pruning to optimize decision-making efficiency. The agent evaluates board states using a heuristic evaluation function that weights positional advantages such as corners and edge stability. I engineered an efficient recursive search procedure with adjustable depth control and pruning to minimize computational overhead while maintaining strategic accuracy. The program consistently outperformed baseline agents using random or sequential move selection, demonstrating the effectiveness of pruning in reducing search complexity. This project strengthened my understanding of game tree exploration, heuristic optimization, and AI search algorithms for adversarial environments.

Constraint Satisfaction and Pandiagonal Magic Squares

This project focused on applying constraint satisfaction techniques to logic-based and mathematical problems using Python’s constraint module. I modelled complex problems — including a traveller scheduling puzzle and the generation of magic and pandiagonal magic squares — as constraint satisfaction problems (CSPs). Through this, I demonstrated strong logical reasoning, formal problem abstraction, and the ability to translate natural language conditions into precise computational rules. The project deepened my understanding of search space reduction, CSP heuristics, and computational logic in AI problem-solving.

Harris Corner Detection and Feature Matching

The objective of this project is to implement the Harris Corner Detector from scratch using Python, demonstrating its role in identifying stable keypoints in computer vision applications. The project includes developing a complete image processing pipeline with reflection padding, Gaussian smoothing, and gradient-based corner response computation. Performance was evaluated under varying illumination, noise, and geometric transformations to test robustness and repeatability. Additionally, ORB descriptors and feature matching using SSD and Lowe’s Ratio Test were applied for comparison. The results highlight the importance of preprocessing and parameter tuning for achieving accurate and reliable feature detection in real-world vision systems.

Natural Language Processing

Comparative Study of Deep Learning Models for Multi-Label Movie Genre Classification

Implemented and compared two advanced NLP architectures for multi-label genre classification of movie plots. The first approach used a hybrid Bidirectional LSTM with Attention and GloVe embeddings, incorporating dynamic class weighting, focal loss, and threshold optimization to improve balance and recall. The second approach fine-tuned a BERT transformer with smooth dynamic class reweighting and learning rate scheduling, leveraging GPU acceleration for efficient training. Both models were trained on preprocessed text data and evaluated using F1-score, precision, and recall, highlighting trade-offs between interpretability and generalization. The project demonstrated strong skills in deep learning optimization, model comparison, and end-to-end NLP pipeline design using TensorFlow, PyTorch, and HuggingFace.

Meta-Learning Approach for Natural Language Inference

Developed in collaboration with a colleague, this project focused on predicting hypothesis–premise relationships within Natural Language Inference (NLI) tasks. Through extensive research on the strengths and limitations of transformer-based models, we explored innovative strategies to improve reasoning accuracy. Building on this analysis, we proposed a meta-learning architecture that aggregates probability outputs from multiple fine-tuned transformers (BERT, RoBERTa, and ALBERT) using a neural meta-learner to enhance generalization and performance. I authored detailed model cards and a poster presentation explaining the design, ethical considerations, and interpretability aspects of both solutions. Additionally, I developed a clean, modular demo codebase that clearly separates training, validation, and inference, emphasizing reproducibility and practical deployment.

Other Research & Innovation Projects

Greening Solution for Environmental Pollution – Converting Waste Clothes into Smart Car Exhaust Filters

I conducted an independent 86-page research project exploring the use of waste textiles and biomass as eco-friendly car exhaust filters to reduce CO, CO₂, and hydrocarbon emissions. Through 24 experiments on different fuel types, I demonstrated that waste-based filters could achieve up to 100% CO, 96% HC, and 79% CO₂ absorption efficiency, providing a sustainable and low-cost solution to both air pollution and textile waste. The project earned multiple international awards, including an Honourable Mention at Genius Olympiad 2019 (SUNY Oswego), Silver Medal at Inofair 2020 (USA), and Silver Medal at Euroinvent 2020 (Romania), recognizing its innovation and environmental impact.
